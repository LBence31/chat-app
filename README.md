# CLI Chat App

_Simple CLI Chat room server and client to demonstrate sockets_

## Goal of the project

To demonstrate socket programming in C with these things in mind:

- set up basic socket connections
- use polling on server
- use polling on client
- encapsulate data in a small header and packet structure (TODO)
- continuously check send and recv to see the whole packet is sent/received (TODO)
- correctly handle and support multi byte chars with error handling (TODO)
